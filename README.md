# Code-Nakshatra-Lost-Found-Web-App# 🔍 Lost & Found Web App

A simple and interactive **Lost & Found Web Application** built using **HTML, CSS, and JavaScript**.
This project allows users to report lost items or post found items, making it easier to reconnect items with their rightful owners.

---

## 🚀 Features

* 📤 Add Lost or Found Items
* 📸 Upload Item Images
* 📝 Detailed Description & Contact Info
* 🔍 Filter Items (Lost / Found / All)
* 🗑️ Delete Items
* 💾 Data stored using **LocalStorage** (no backend required)
* 📱 Responsive Design (works on mobile & desktop)

---

## 🛠️ Technologies Used

* **HTML5** → Structure of the application
* **CSS3 (Flexbox)** → Styling and layout
* **JavaScript (Vanilla JS)** → Functionality and logic
* **LocalStorage API** → Data persistence in browser
* **FileReader API** → Image upload and preview

## ⚙️ How It Works

1. User fills the form with item details
2. Uploads an image (optional)
3. JavaScript reads the data
4. Image is converted to Base64 format
5. Data is saved in browser using LocalStorage
6. Items are displayed as cards on the screen

---

## ▶️ How to Run the Project

1. Download or clone this repository
2. Open the project folder in **VS Code**
3. Open `index.html`
4. Run using **Live Server** or open in browser

---

## 📸 Demo Usage

* Click on **Choose File** to upload an image
* Fill item details (name, location, description, contact)
* Select **Lost** or **Found**
* Click **Submit**
* Item will appear in the list

---

## ⚠️ Limitations

* Data is stored locally (not shared online)
* Images are stored in browser memory (Base64)
* Data will be lost if browser storage is cleared
